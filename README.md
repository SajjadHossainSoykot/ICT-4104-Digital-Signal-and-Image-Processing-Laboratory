# 🖼️ Digital Signal and Image Processing Laboratory

This repository contains practical implementations and laboratory experiments developed as part of the course:

**ICT-4104 – Digital Signal and Image Processing Laboratory**

---

## 📘 Course Overview

This laboratory course focuses on the practical implementation of digital image processing techniques using programming tools. It emphasizes how images are represented, processed, enhanced, and analyzed in real-world applications.

The course bridges theoretical concepts of signal and image processing with hands-on coding experience to build a strong foundation in visual data analysis.

### Key Topics Covered

- Image representation (RGB & Grayscale)  
- Image gradient and edge detection  
- Histogram analysis  
- Image transformation (negative)  
- Image enhancement (brightness, contrast, color)  
- Image sharpening techniques  
- Image smoothing (Average, Gaussian, Median filters)  
- Spatial domain processing  

---

## 🧪 Laboratory Experiments

This repository includes implementations of the following experiments:

1. Image Gradient Calculation  
2. Histogram Visualization  
3. Negative Image Transformation  
4. Color Enhancement  
5. Brightness Enhancement  
6. Contrast Enhancement  
7. Image Sharpening  
8. Average Filtering (Smoothing)  
9. Gaussian Filtering (Smoothing)  
10. Median Filtering (Noise Reduction)  

---

## 📂 Repository Contents

- Jupyter Notebook (`.ipynb`) files for each experiment

## 🚀 How to Run

### 1. Clone the Repository

    git clone https://github.com/SajjadHossainSoykot/ICT-4104-Digital-Signal-and-Image-Processing-Laboratory.git
    cd ICT-4104-Digital-Signal-and-Image-Processing-Laboratory

---

### 2. (Optional) Create Virtual Environment

    python -m venv venv

Activate environment:

- Windows:

    venv\Scripts\activate

- Linux/Mac:

    source venv/bin/activate

---

### 3. Install Required Libraries

    pip install opencv-python numpy matplotlib

---

### 4. Run Python Scripts

    python filename.py

---

### 5. Run Jupyter Notebook

    jupyter notebook
